# Tesla Cycle Timing Indicator (TSLA)

## Install (30 seconds)
1. Open the script file below
2. Click **Copy raw file**
3. TradingView → **Pine Editor**
4. Paste → **Add to chart**

## Script
➡️ **Open and copy this file:**
- `tesla_cycle_timing.pine`
